# 运行日志

系统异常退出或内存偏高时，会写入此目录：

| 文件 | 内容 |
|------|------|
| `supervisor.log` | 守护进程启动/子进程退出 code |
| `backend-YYYY-MM-DD.log` | 主后端未捕获异常、内存告警、启动失败 |

**若再次「挂掉」**：把崩溃当天这两个文件发给维护人员，可定位是窗口被关、OOM 还是代码异常。

常见非崩溃原因：

- 关闭了 `start.bat` 黑色窗口 → 全部服务停止
- 电脑休眠/重启/Windows 更新
- Excel 被关掉后 Excel 桥接反复重启（主服务仍在，但 Excel 同步会失败）
